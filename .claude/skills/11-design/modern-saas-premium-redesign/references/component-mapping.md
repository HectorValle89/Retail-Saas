# Component Mapping

## Card

- Fondo blanco o marfil frio
- Sin borde duro por defecto
- Radius 20px
- Sombra difusa, baja opacidad
- Padding amplio

## Button

- Primary: verde esmeralda, leve gradiente permitido, texto blanco, radio 14px
- Secondary: blanco o gris muy suave, borde sutil, texto oscuro
- Ghost: hover muy sutil
- Danger: rojo suave pero premium, no agresivo

## Input / Select / Textarea

- Fondo `#F8FAFC`
- Borde `#E2E8F0`
- Radius 12px
- Labels arriba, pequenas y elegantes
- Focus verde con halo translucido

## Badge / Pill

- Pastel + texto fuerte
- Siempre en capsula completa
- Nada de grises sin intencion

## Sidebar

- Fondo blanco limpio
- Activo en pill verde suave
- Icono outline + label
- Hover suave

## Header / Hero

- Tarjeta premium blanca, no bloque negro pesado
- Puede usar borde izquierdo verde o gradiente muy leve
- Debe sentirse como bloque de liderazgo visual

## Tabla

- Reducir gridlines
- Headers discretos
- Filas con espacio y tratamiento soft
- Estados como pills

## Formulario

- Grupos con mejor separacion vertical
- Cards limpias
- Acciones principales con peso visual claro
