# Visual Direction

## Intencion

Llevar la app de una estetica tecnica-administrativa a una estetica Modern SaaS Premium, luminosa, ejecutiva y suave, inspirada por TeamHub pero adaptada al contexto operativo retail.

## Fundaciones visuales

- Fondo app: `#F8FAFC`
- Surface principal: `#FFFFFF`
- Surface secundaria: `#F1F5F9`
- Texto primario: `#0F172A`
- Texto secundario: `#475569`
- Texto tenue: `#64748B`
- Primario: `#2CB67D`
- Primario fuerte: `#1F8F63`
- Primario suave: `#E8F8F1`
- Borde sutil: `#E2E8F0`

## Tipografia

- Base: `Inter`
- Heading accent: `Poppins`
- Titulo de pagina: grande, semibold
- Titulo de tarjeta: 18-20px, semibold
- KPIs: 32-40px, bold
- Labels: 12-13px, medium, tono secundario

## Elevacion y superficies

- Tarjetas sin borde dominante.
- Radius base de tarjeta: `20px`
- Sombra base: `0 10px 30px rgba(15,23,42,0.05)`
- Sombra hover: `0 18px 40px rgba(15,23,42,0.08)`
- Mucho padding interno; el producto debe respirar.

## Iconografia

- Outline, 24px, trazo fino/moderno
- Usar circulos pastel para KPI/icon chips
- Nunca mezclar outline minimalista con iconos solidos pesados

## Estados

- Success: verde suave
- Pending: ambar suave
- Error: rojo pastel
- Info: cyan/sky suave
- Focus: halo verde translúcido, visible pero elegante
